##CM3D2.AddYotogiSlider.Plugin (ANALKUPA, AutoTUN 拡張 Ver.)

夜伽コマンド画面中にF5でGUI表示トグル。  
夜伽中のメイドのステータス・表情等をスライダー・ボタンで変更する事が可能。  
![GUI](http://i.imgur.com/KNSFUPR.png "GUI")  



##導入方法

**前提条件** : **UnityInjector** が導入済みであること。  
  
[![ダウンロードボタン][img_download]][master zip]を押してzipファイルをダウンロード。  
zipファイルの中にあるUnityInjectorフォルダを CM3D2\Sybaris\Plugins フォルダにD&Dすれば導入完了。  

ANALKUPA 機能を使うには **'analkupa' 対応 body が必要です。**
対応 body がなくてもそれ以外の機能は動作します。
また、labiakupa, vaginakupa, nyodokupa, sujiをスライダーで変化させることができます。
clitoris に対応しているbodyではクリが興奮によって変化します。
AutoTUN  機能を使うには **'chikubi_bokki' 対応乳首 が必要です。**
対応乳首がなくてもそれ以外の機能は動作します。

しばりす環境以外での動作確認はしていません。
本体1.31で動作確認しています。

##AutoTUN
AutoTUN  機能を使うには **'chikubi_bokki' 対応乳首 が必要です。**
（@30767_cm3d2 氏配布のぼっき乳首をメイドさんに適用します）
http://ux.getuploader.com/futanarichinko/download/65/30767_cbokki.zip

乳首肥大度     : 乳首勃起の比率です。デフォルトは25です。数値を大きくすると最大サイズの大きい肥大乳首になります。
乳首萎え       : 夜伽を始めた直後の値を設定します。デフォルトは0です。マイナスの値を指定すると（元のぼっき乳首MODには無い）微陥没乳首になるように「乳首たれ」を調節します。
乳首勃起       : 乳首の現在値です。ゲーム中、乳首を刺激する行動をとった場合に少しずつ上昇します。「パイズリ」など一部の夜伽スキルでは停止以外で常に上昇します。また、夜伽スキル切り替え時に半減します。
乳首たれ       : 乳首が足元方向に向かって首をかしげるシェイプキーです。基本は固定ですが、乳首の大きさにマイナスを指定した場合、微陥没乳首機能が優先されてこの値が上書きされます。

##AutoKUPA
ANALKUPA 機能を使うには **'analkupa' 対応 body が必要です。**
対応 body がなくてもそれ以外の機能は動作します。対応 body のリリース時期によって前穴しかないものからvaginakupaがあるものまで何種類かバージョンがありますが、シェイプキーの存在有無を判別して動作を決定しています。
